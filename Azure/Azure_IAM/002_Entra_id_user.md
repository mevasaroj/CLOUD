# 1. Entra ID User
 - A user in Microsoft Entra ID (formerly Azure Active Directory) is a digital identity representing an individual in organization's cloud-based directory.

## 2. Types of Users
 - A Microsoft Entra workforce tenant has the following user types:
    - **Internal member**: These users are most likely full-time employees in your organization.
    - **Internal guest**: These users have an account in your tenant, but have guest-level privileges.
    - **External member**: These users authenticate using an external account, but have member access to your tenant. These types of users are common in multitenant organizations.
    - **External guest**: These users are true guests of your tenant who authenticate using an external method and who have guest-level privileges.

## 3. Create a new user
 - Sign in to the **Microsoft Entra admin center** as at least a *User Administrator*.
 - Type **Entra ID** in search bar --> Click on **Microsoft Entra ID**
 - Click **Add** --> Select **User** --> Click **Create new user**
 - Under **Basic**
    - User principal name = type name like **msaroj** --> Select domain from drag menu.
    - Display name = Mevalal Saroj
    - Password = Type the Password
    - Account Enabled = Tick Mark
    - Click **Next: Properties** Button
 - Under **Properties**
    - First name = Mevalal
    - Last name = Saroj
    - User type = Member
    - Fill All Others Information
    - Click **Next: Assignment** Button 
 - Under **Assignment**
    - Click on **+ Add Role** --> Assign Require Permision --> Click **Select**
    - Click **Next: Review + Create** Button
 - Under **Review + Create** --> Review the information --> Click **Create**
